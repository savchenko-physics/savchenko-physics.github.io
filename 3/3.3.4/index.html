Условие:

$3.3.4.$ Частота свободных колебаний тела равна $ω$. Через какое наименьшее время его кинетическая энергия уменьшается вдвое по сравнению со своим наибольшим значением?

Решение:

Из 3.3.1, мы выяснили что скорость зависит от времени по закону $$v(t)=v_0\sin\omega t$$ Учитывая, что кинетическая энергия определяется выражением $$E_k = \frac{mv^2}{2}$$ Зависимость кинетической энергия от времени будет иметь вид $$E_k = E_0 \sin^2\omega t$$ Момент времени, когда $E_k$ составит $E_k = \frac{E_0}{2}$ опишется уравнением $$\sin^2\omega t = \frac{1}{2}$$ Откуда $$\sin\omega t = \frac{\sqrt{2}}{2}$$ Это произойдет в ближайший момент равный $$t = \frac{π}{4ω}$$

Ответ:

$$t = \frac{π}{4ω}$$
